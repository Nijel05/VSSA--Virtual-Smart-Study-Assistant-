## 📖 About the Project

The **Virtual Smart Study Assistant (VSSA)** was developed as my Senior Capstone Project while completing my Bachelor's degree in Software Engineering at **Norfolk State University**.

The goal of VSSA is to create an organized and interactive study platform that helps students prepare for exams more efficiently. Instead of managing notes across multiple apps, students can build custom study sessions, create flashcards, organize practice questions, and review material using one centralized application.

This project allowed me to gain real-world experience working on a collaborative software engineering team while applying full-stack development principles from planning and design through implementation and testing.

---

# ✨ Features

- 🔐 Secure user authentication
- 📖 Create and manage study questions
- 📝 Build personalized study sessions
- ⏱️ Built-in Pomodoro-style study timer
- 🎴 Flashcard support
- 📊 Practice quizzes
- 🎯 Randomized study questions
- 📂 Import questions from text files
- ❌ Delete and manage study content
- 💾 Persistent data storage
- 📱 Responsive interface

# 🏗️ Project Architecture

The application follows a full-stack architecture consisting of:

```
Frontend (React)
│
▼
REST API (Node + Express)
│
▼
MySQL Database
```

The frontend communicates with the backend using RESTful API endpoints while the backend manages authentication, study content, and database interactions.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Nijel05/VSSA--Virtual-Smart-Study-Assistant-.git
```

Navigate into the project

```bash
cd VSSA--Virtual-Smart-Study-Assistant-
```

Install dependencies

```bash
npm install
```

Start the application

```bash
npm run dev
```
